# SAI ホームページ

[Next.js](https://nextjs.org/)ベースのSAIホームページ

## 開発

クローンの後，下記のいずれかを実行

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## 現在使用している機能

- [GAS(Google Apps Script)](https://workspace.google.co.jp/intl/ja/products/apps-script/)
- [Vercel](https://vercel.com/)
